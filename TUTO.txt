Pour Tomcat :
	Vérifier que dans le ApplicationContext.xml c'est bien "localhost:3306"
	Build Maven
	docker build -t mon-projet-java . # Build le fichier issue de la compilation Maven
	docker network rm asbank_network # Effacer le réseau s'il existait précédemment (utile plus tard pour Docker, mais nécessaire pour docker-compose up -d)
	docker network create asbank_network # Recréer le réseau
	docker-compose up -d # Lancer tout le bazar
	Aller sur localhost:8082 -> la BDD doit apparaitre, aller sur testdb et importer les données (parfois un F5 est nécessaire)
	Lancer Tomcat, normalement ça fonctionne

Pour Docker :
	Vérifier que dans le ApplicationContext.xml c'est bien "db:3306"
	Build Maven
	docker build -t mon-projet-java . # Build le fichier issue de la compilation Maven
	docker network rm asbank_network # Effacer le réseau s'il existait précédemment (nécessaire pour docker-compose up -d)
	docker network create asbank_network # Recréer le réseau
	docker-compose up -d # Lancer tout le bazar
	Aller sur http://localhost:8082 -> la BDD doit apparaitre, aller sur testdb et importer les données (parfois un F5 est nécessaire)

	# Lance mon-projet-java sur le bon port, avec le bon nom, et les variables d'environnement + connecte le container au même réseau que la BDD
	docker run -d `
		--name mon-projet-java `
		--network asbank_network `
		-p 8080:8080 `
		-e MAIL_USER="ton_email@gmail.com" `
		-e MAIL_PASSWORD="ton_mot_de_passe_app" `
		mon-projet-java

	OU

	Sur Docker Desktop, lancer mon-projet-java (image, pas le container) avec comme port 8080:8080 et comme nom de container "mon-projet-java", et ajouter 2 variables d'environnement :
		- MAIL_USER : Le mail à utiliser
		- MAIL_PASSWORD : Le mot de passe d'application associé à ASBank
	docker network connect asbank_network mon-projet-java # Permet de connecter le container là au même réseau que la BDD

	Aller sur http://localhost:8080/_00_ASBank2023/ normalement ça fonctionne

Si vous voulez arrêter les trucs Docker pour changer par exemple : docker-compose down -v
